개인정보처리방침

시행일: 2026-04-21

‘쓰레기봉투 어디 있어?’(이하 ‘서비스’)는 이용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수합니다.

1. 수집하는 정보

  • 서비스는 회원가입 기능이 없으며, 이름/전화번호/이메일/주민등록번호 등 식별정보를 직접 수집하지 않습니다.
  • 위치 기반 검색 기능 사용 시, 기기에서 제공한 위치정보(위도/경도)를 사용합니다.

2. 정보 이용 목적

  • 내 주변 종량제봉투 판매처 검색
  • 지역별 가격·판매처 정보 조회
  • 사용자 재고 제보(있음/없음) 표시

3. 정보 저장 및 보관

  • 재고 제보 상태는 기기 로컬 저장소(localStorage)에 저장될 수 있습니다.
  • 민감한 개인정보(계정 비밀번호, 결제정보 등)는 저장하지 않습니다.

4. 제3자 서비스 이용

서비스는 아래 외부 서비스를 사용합니다. 각 서비스 제공자에게 접속 정보(IP, User-Agent, 요청 시각, 위치정보 등)가 처리될 수 있습니다.

  • Kakao Maps SDK / Places / Geocoder
  • 공공데이터포털(Open API)
  • GitHub Pages(정적 파일 호스팅)

5. 이용자 권리

  • 기기 설정에서 위치 권한을 언제든지 허용/차단할 수 있습니다.
  • 브라우저/웹뷰 데이터 삭제 시 로컬 저장 데이터가 함께 삭제될 수 있습니다.

6. 보안 조치

  • 외부 API 키는 서비스 도메인 제한 등 접근 통제를 적용합니다.
  • 필요 최소 범위의 데이터만 처리하도록 운영합니다.

7. 방침 변경

본 방침은 관련 법령 또는 서비스 변경 시 수정될 수 있으며, 변경 시 본 페이지에 공지합니다.
